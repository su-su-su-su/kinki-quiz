# Kinki Quiz

近畿地方の県庁所在地に関するクイズをランダムに提供する、シンプルなnpmパッケージです。

## インストール

```bash
npm install kinki-quiz
```

## 使い方

このパッケージを使って作成した簡単なクイズアプリ (run-quiz.js) を実行すると、次のように対話形式でクイズができます。

```bash
$ node run-quiz.js
問題: 奈良県の県庁所在地はどこでしょう？
> 奈良市
正解です！
```

```bash
$ node run-quiz.js
問題: 兵庫県の県庁所在地はどこでしょう？
> 大阪市
不正解です。正しい答えは「神戸市」です。
```
